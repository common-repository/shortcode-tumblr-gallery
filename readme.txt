Tumblr Gallery - Display Photos or Videos from your Tumblr account

== Description ==

Tumblr Gallery display all your photos and videos on your site.
 You can insert shortcode in post, page or sidebar, widget.

== Installation ==

<h4>1. Automatic installation:</h4>

The simplest way to install is to click on 'Plugins' then 'Add new' and type 'Tumblr Gallery' in the search field.

<h4>2. Manual installation 1:</h4>
<ol>
<li>Login to your website and go to the Plugins.</li>
<li>Click the Add New button.</li>
<li>Under Install Plugins, click the Upload Plugin.</li>
<li>Select the plugin zip file (tumblr-gallery.zip) from your computer then click the Install Now button.</li>
<li>You should see a message stating that the plugin was installed successfully.</li>
<li>Click the Activate Plugin.</li>
</ol>

== Frequently Asked Questions ==

= How to display Tumblr  =
Please copy shortcode [tumblr_gallery id="1"] and paste on your post or page, sidebar, widget.
